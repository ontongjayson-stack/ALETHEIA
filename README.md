# ALETHEIA
### *ἀλήθεια — Truth Cannot Be Hidden*

> A permanent, decentralized archive of suppressed inventions, free energy devices, sustainable technologies, and open-source blueprints. Once published here, no corporation can erase it.

---

## What Is ALETHEIA?

Throughout history, powerful institutions have suppressed technologies that threatened their control.

The water-powered engine. The free energy device. The cure that never made it to market. The inventor who disappeared.

They could kill a person. They could buy a patent. They could burn a lab.

**But they cannot kill an idea shared with everyone, simultaneously, permanently, on a network no one controls.**

ALETHEIA is a decentralized knowledge archive — like GitHub, but for inventions, schematics, blueprints, and instructions for technologies that the mainstream system buried. Built on blockchain infrastructure so that once a blueprint is published, it exists forever. No takedown notice. No buyout. No server to seize.

---

## Features

- **Blueprint Archive** — Searchable database of suppressed and open-source inventions across energy, water, medical, transport, and more
- **AI Brain** — Powered by Claude. Ask any question about suppressed technology and get researched, sourced answers
- **Scraper Engine** — Pulls publicly available schematics and patents from 12+ open databases including Google Patents, USPTO, Espacenet, Archive.org, arXiv, and Open Source Ecology
- **Submit System** — Any inventor can submit blueprints for moderation review before permanent on-chain publication
- **Manifesto** — The philosophy behind the archive

## The Four Pillars

**01 — PERMANENCE**
All verified submissions are written to Arweave — a permanent, decentralized storage network. No server can be seized. No domain can be taken down.

**02 — OPENNESS**
Every blueprint, schematic, and instruction set is freely accessible to every human on earth. No paywalls. No gatekeepers. Knowledge belongs to everyone.

**03 — PROTECTION**
A decentralized council of moderators reviews submissions for quality and safety before publication. Content is filtered for harm — not for inconvenience to the powerful.

**04 — TRUTH**
*ἀλήθεια* — the ancient Greek concept of truth as that which cannot remain concealed. That is what ALETHEIA does. It unconceals. It reveals. It publishes what was buried.

## Getting Started

No installation required. ALETHEIA runs as a single HTML file.

1. Clone this repository
2. Open `aletheia.html` in any browser
3. The platform is fully functional

To enable the AI Brain, add your [Anthropic API key](https://console.anthropic.com).
